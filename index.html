06
Functions & Scope
pass-by-value · scope · recursion · function declarations · blocks
CFunctionsRecursion
Function Pattern
/* Declaration */
int factorial(int n);

/* Recursive definition */
int factorial(int n) {
    if(n <= 1) return 1;
    return n * factorial(n-1);
}

/* Fibonacci recursive */
int fib(int n) {
    if(n<=1) return n;
    return fib(n-1) + fib(n-2);
}
Scope & Pass-by-Value
  • Pass-by-value: function gets a copy — changes don't affect the original in caller
  • Block scope: variables in { } live only within that block
  • Uninitialized locals have garbage values
  • Swap via pass-by-value does NOT work in caller
  • Return statement after another return is dead code — never executes
07
Arrays
linear search · binary search · selection sort · pass-by-reference
CArraysSorting
KEYArrays are passed by reference — changes inside function affect the original. sizeof(arr) inside a function = 8 (pointer size), NOT array size. Always pass size as a parameter.
Linear & Binary Search
/* Linear O(n) — any array */
int linSearch(int a[], int n, int x) {
    for(int i=0; i<n; i++)
        if(a[i]==x) return i+1;
    return -1;
}

/* Binary O(log n) — SORTED ONLY */
int binSearch(int a[], int n, int x) {
    int lo=0, hi=n-1;
    while(lo<=hi) {
        int mid=(lo+hi)/2;
        if(a[mid]==x) return mid+1;
        else if(a[mid]<x) lo=mid+1;
        else hi=mid-1;
    }
    return -1;
}
Selection Sort (Decreasing)
void selSortDec(int a[], int n) {
    for(int i=0; i<n-1; i++) {
        int mx = i;
        for(int j=i+1; j<n; j++)
            if(a[j]>a[mx]) mx=j;
        int t=a[i]; a[i]=a[mx]; a[mx]=t;
    }
}
/* Print ascending: loop backwards */
for(int i=n-1; i>=0; i--)
    printf("%d ", a[i]);

Linker Errors
  • Undefined reference to 'foo': forgot to link .o file containing foo()'s definition
  • Multiple definition of 'foo': two .o files both define the same function
  • Undefined reference to main: main .o not included, or typo Main() vs main()
  • Only ONE main() allowed across all linked files
  • .h = declarations only; .c = implementations

int *ptr = &x;    // ptr = address of x

/* Pointer arithmetic on arrays */
int arr[] = {22,54,78};
int *p = arr;
*(p++)  // 22, then p moves to arr[1]
*++p    // moves FIRST, gives arr[1]=54
*p+5    // value at p + 5 = 27
arr[1] == *(arr+1) == *(p+1)
Call-by-Reference

    *area = 3.14159 * r * r;
    *peri = 2 * 3.14159 * r;
}
/* Call: */
double a, p;
circleData(5.0, &a, &p);

/* Arrays as pointers (same!) */

Dynamic Allocation
#include <stdlib.h>

/* malloc: uninitialized */
int *arr = (int*)malloc(n * sizeof(int));

/* calloc: zero-initialized */
int *arr = (int*)calloc(n, sizeof(int));

/* Always check NULL */
if(arr == NULL) { /* handle error */ }

arr[0] = 10;  // use like normal array

/* ALWAYS free heap memory */
free(arr);
Command Line Arguments
int main(int argc, char *argv[]) {
    // argc = count (includes prog name)
    // argv[0] = program name "./a.out"
    // argv[1] = first arg (string)
    
    int n = atoi(argv[1]); // str→int
}
/* ./prog 42 hello → argc=3 */
/* argv[1]="42", argv[2]="hello" */
WARNHeap memory is NOT auto-released. Always free(ptr). Never dereference uninitialized pointer: int *p; *p = 5; is UB (crash). Always initialize or malloc first.

11M
▼

Right Rotate Array by K Positions

EASY
▼ Show Solution
Problem
Solution
Key Insight
You are given an array of integers arr of size n and an unsigned integer rotateBy. Implement the function void rotateArray(int arr[], int n, int rotateBy) that rotates the array to the right by rotateBy positions in-place.
Sample 1: arr=[1,2,3,4,5], rotateBy=2 → [4,5,1,2,3] Sample 2: arr=[10,20,30,40], rotateBy=18 → [30,40,10,20] Sample 3: arr=[1,2,3,4,5], rotateBy=0 → [1,2,3,4,5]
Hint: Use modulo to handle rotateBy > n. Then use the three-reversal trick: reverse the whole array, reverse the first k elements, reverse the last n-k elements.
f1.cImplementation
void reverse(int arr[], int lo, int hi) { while(lo < hi) { int tmp = arr[lo]; arr[lo++] = arr[hi]; arr[hi--] = tmp; } } void rotateArray(int arr[], int n, int rotateBy) { /* Handle rotateBy larger than n */ rotateBy = rotateBy % n; if(rotateBy == 0) return; /* Three-reversal trick */ reverse(arr, 0, n-rotateBy-1); // reverse first n-k reverse(arr, n-rotateBy, n-1); // reverse last k reverse(arr, 0, n-1); // reverse whole }
Key Insights
  • Right rotate by k = last k elements go to front. So [1,2,3,4,5] rotated right 2 = [4,5,1,2,3]
  • Three-reversal works because reversing partitions and then the whole reverses the rotation
  • Always normalize: k = rotateBy % n before anything else — handles k > n
  • Alternative O(n) extra space method: copy last k to temp, shift array right by k, copy temp to front

void pattern(int n) { /* Top half: rows 1 to n */ for(int i = 1; i <= n; i++) { /* Leading spaces */ for(int j = 0; j < n-i; j++) printf(" "); if(i == 1) { printf("*"); // top tip } else { printf("*"); for(int k = 0; k < 2*i-3; k++) printf(" "); printf("*"); } printf("\n"); } /* Bottom half: rows n-1 down to 1 */ for(int i = n-1; i >= 1; i--) { for(int j = 0; j < n-i; j++) printf(" "); if(i == 1) { printf("*"); } else { printf("*"); for(int k = 0; k < 2*i-3; k++) printf(" "); printf("*"); } printf("\n"); } }

  • Total rows = 2n-1. Middle row is row n (widest)

  • arr+i = pointer to row i. *(arr+i) = pointer to first element of row i. *(*(arr+i)+j) = element at row i, col j
  • This is EXACTLY equivalent to arr[i][j] — the compiler generates identical code
  • 2D arrays in C are stored row-major: row 0 entirely, then row 1, etc. — contiguous in memory
  • For half-pyramid: inner loop goes from j=0 to j≤i — row 0 prints 1 element, row 1 prints 2, etc.
  • Function signature must have column size: int arr[N][N] or int arr[][N] — never just int **arr

The program receives the number of employees as a command-line argument. Dynamically allocate an array of Employee structs (fields: id int, name char[30], salary float). Read all employees, find the one with the highest salary, print all records and the top earner. Free all memory before exit.

Hint: malloc(n * sizeof(Employee)). Access fields via emp[i].salary. Free with a single free(emp) at the end.
main.cFull Solution
#include <stdio.h> #include <stdlib.h> typedef struct { int id; char name[30]; float salary; } Employee; int main(int argc, char *argv[]) { if(argc != 2) { printf("Usage: %s <n>\n", argv[0]); return 1; } int n = atoi(argv[1]); /* Allocate array of structs */ Employee *emp = (Employee*)malloc(n * sizeof(Employee)); if(!emp) { printf("Alloc failed\n"); return 1; } for(int i=0; i<n; i++) scanf("%d %s %f", &emp[i].id, emp[i].name, &emp[i].salary); printf("\nAll employees:\n"); for(int i=0; i<n; i++) printf("%d %s %.2f\n", emp[i].id, emp[i].name, emp[i].salary); /* Find top earner */ int top = 0; for(int i=1; i<n; i++) if(emp[i].salary > emp[top].salary) top = i; printf("Top earner: %s (%.2f)\n", emp[top].name, emp[top].salary); free(emp); /* single free for the whole array */ return 0; }
Key Insights
  • malloc(n * sizeof(Employee)) allocates a contiguous block for n structs — access with emp[i].field like any array
  • Note: scanf("%s", emp[i].name) — no & for char arrays because array name IS a pointer
  • A single free(emp) frees the entire block — only one malloc → only one free
  • Always: argc check → atoi → malloc → NULL check → use → free. This is the full lab 11 pattern
  • When struct is accessed via pointer: use emp[i].salary (array indexing) or (emp+i)->salary — both correct
